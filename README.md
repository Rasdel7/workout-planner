# Personal Workout Planner 🏋️

Personalized weekly workout plans, exercise
tracking and fitness progress analytics.

## Live Demo
[Click here](YOUR_STREAMLIT_URL)

## Features
- 4 plan templates (Beginner to Advanced)
- Daily workout view based on today's day
- 35+ exercises across 7 muscle groups
- Calorie burn estimation by weight
- Workout logging with intensity tracker
- Progress charts: calories, duration, frequency
- Exercise library with filters
- Equipment-aware (gym / bodyweight / dumbbells)

## Tools Used
- Python, Streamlit, Plotly, Pandas, NumPy

## How to Run Locally
pip install streamlit pandas numpy plotly
streamlit run app.py
